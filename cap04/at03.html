3. Glossário Técnico

HTML
Linguagem de marcação para criação de páginas web.
CSS
Linguagem de estilos para formatação visual.
JS
JavaScript
Linguagem de programação interpretada de alto nível.
API
Interface de Programação de Aplicações.
Conjunto de normas que possibilitam a comunicação entre plataformas.
HTTP
HTTPS
Protocolo de transferência de hipertexto entre cliente e servidor.
DOM
Document Object Model.
Interface que representa documentos HTML como uma estrutura de árvore.
URL
Localizador Padrão de Recursos na rede.
DNS
Sistema que traduz nomes de domínios em endereços IP.
